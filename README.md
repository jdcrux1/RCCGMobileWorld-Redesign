# RCCG Mobile World Redesign

Official website for The Redeemed Christian Church of God (RCCG). A modern, responsive web platform for worship services, sermons, events, and community engagement.

## Features

- Live streaming of services
- Sermon archive with playlist
- Event management and registration
- Parish locator
- Online giving integration
- Contact forms and inquiries
- Responsive design (mobile, tablet, desktop)
- Dark and light theme variants

## Tech Stack

- HTML5 / CSS3
- Tailwind CSS (via CDN)
- Vanilla JavaScript
- YouTube API (video embedding)

## Getting Started

Simply open `index.html` in your browser, or deploy to any static hosting provider:

```bash
# View locally
open index.html
```

## Deployment

This project deploys seamlessly to Vercel:

1. Push to GitHub
2. Import project in Vercel
3. Deploy with default settings
